## ğŸ’¡ How to Use These Prompts

### 1. Choose Your Tool

Most prompts work in:
- **Claude Projects** - Best for complex, multi-turn workflows
- **ChatGPT Projects** - Great for iterative work with memory
- **Cursor / Copilot** - For code-related prompts
- **NotebookLM / Gemini** - For document analysis
- **Lovable / v0 / Bolt** - For rapid prototyping

### 2. Customize for Your Context

Each prompt includes placeholders like `[YOUR_GOAL]` - replace with your specifics.

**Example:**
```
Bad:  "I need a PRD"
Good: Paste the PRD prompt + add your specific context
```

### 3. Iterate

These are starting points, not magic bullets:
1. Run the prompt
2. Review output
3. Add context or clarify
4. Refine until perfect
